# Install
```sh
npm init slidev@latest
```
---

# Basic Structure
A Slidev presentation is primarily written in a single Markdown file, typically `slides.md`. Slides are separated by three dashes `(---)`.

```markdown
# Welcome to Slidev

Your first presentation

---

# Page 2

* Bullet points
* Made easy

---
```
