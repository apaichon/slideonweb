## Versioning Strategies
![Versioning](./images/ch1/1.3.10.svg)
### 1. Version Types
1. **URI Versioning**
   ```text
   Examples:
   /api/v1/users
   /api/v2/users
   ```

2. **Header Versioning**
   ```text
   Accept: application/vnd.company.api+json;version=1
   Accept: application/vnd.company.api+json;version=2
   ```

3. **Parameter Versioning**
   ```text
   /api/users?version=1
   /api/users?version=2
   ```
